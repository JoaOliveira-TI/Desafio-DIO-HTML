Estrutura Básica do HTML

A estrutura básica de um arquivo HTML é o primeiro passo para entender como funciona cada Tag que estrutura a forma como uma pagina é formada. Quando criamos um arquivo HTML, por padrão colocamos o nome de "Index.html" é uma forma de normalizar o nome do arquivo. A estrutura do HTML consiste em: <"html">, <"head">, <"body">, de uma forma bem básica seria isso, cada uma dessas tags devem ser acompanhadas de seus fechamentos (falaremos disso mais a frente). Dentro de <"html"> colocamos todo o conteúdo do html, tando o head e o body ficam dentro desta Tag. O <"head"> é onde ficam as informações de cabeçalho, como o título, e também é aqui que as primeiras informações da pagina são carregadas. O <"body"> é onde colocamos todo o corpo da pagina, todas as informações ficaram no body.

Falando sobre Tags

As tags são as estruturas que definem o conteúdo de uma página HTML. Ela geralmente tem abertura e fechamento, como a teg <"P"> ela serve para colocarmos paragrafo em nossa pagina quando abrimos a teg P devemos fecha-lá da seguite maneira <"p"> Tudo que estiver dentro deste escopo é um paragrafo <"/p">. E é assim como varias outras tegs funcionan, algumas não tem fechamento no caso da tag <"img">.

Atributos das Tags

Os atributos são informações adicionais que podem ser passadas para as tags. Eles são em alguns casos expecificos para algumas Tgas, podemos usar alguns atributos para usar CSS no HTML e assim conseguimos formatar os textos. Os atributos ficam depois da declaração das Tags, como <"input type = "text" />" o type é o atributo de input.

Textos

Os textos são uma das formas mais comuns de se usar o HTML, podemos usar as Tags para estruturar os textos e os atributos para formatar e modificar. Temos as Tags <"H1"> até a <"H6"> elas representam o titulo, sub-título e assim por diante, o H1 é o de maior texto e o H6 é o de menor texto. O <"p"> usamos para paragrafo, e varias outras consigurações.

Listas Ordenadas e Não Ordenadas

As listas são uma das formas mais comuns de se usar o HTML, podemos usar as listas para organizar as informações, temos as lista ordenadas (<"ol">) e as listas não ordenadas (<"ul">). Dentro da nossa lista colocamos a tag <"li"> que representa o item da lista, e assim podemos criar uma lista com varios itens.
exemplo de uma lista não ordenada:
<"ul">
<"li> Estrutura Básica do HTML<"/li>
<'li> Falando sobre Tags<'/li>
<'li> Atributos das Tags<'/li>
<'li> Textos<'/li>
<'li> Listas Ordenadas e Não Ordenadas<'/li>
<'li> Links<'/li>
<"/ul">
